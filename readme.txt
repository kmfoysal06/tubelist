=== Description ==
a simple plugin to load and display youtube playlists through shortcode with the help of youtube v3 api.

=== Installation ===
1. Upload the `tubelist` Folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit TubeList From Dashboard Settings and Set Your API KEY of youtube v3 api.

== Usage ==
1. use [tubelist id="playlistid"] shortcode to load a playlist from youtube.
